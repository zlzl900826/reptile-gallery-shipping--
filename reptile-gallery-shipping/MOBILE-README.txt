렙타일갤러리 배송 신청 사이트 - 핸드폰 접속 안내

1. 컴퓨터와 핸드폰을 같은 와이파이에 연결하세요.
   - 핸드폰이 LTE/5G이면 접속되지 않습니다.

2. 압축 푼 폴더에서 start-windows.bat를 더블클릭하거나,
   명령 프롬프트에서 아래를 실행하세요.

   npm install
   npm start

3. 검은 창에 아래처럼 주소가 표시됩니다.

   컴퓨터 접속: http://localhost:3000
   핸드폰 접속:
     http://192.168.x.x:3000
   관리자 페이지:
     http://192.168.x.x:3000/admin.html

4. 핸드폰 크롬/사파리 주소창에 핸드폰 접속 주소를 그대로 입력하세요.

5. 접속이 안 될 때 확인할 것
   - 컴퓨터와 핸드폰이 같은 와이파이인지 확인
   - npm start 검은 창이 켜져 있는지 확인
   - Windows 방화벽에서 Node.js / node.exe 개인 네트워크 허용
   - 회사/공유기 게스트 와이파이는 기기 간 접속이 막혀 있을 수 있습니다.

6. 외부 손님도 각자 집에서 접속하게 하려면
   - 로컬 실행이 아니라 Render 같은 서버에 배포해야 합니다.
   - Render에 배포하면 https://원하는이름.onrender.com 주소로 컴퓨터/핸드폰 모두 접속 가능합니다.
