# 렙타일갤러리 낙찰 결제/수령 신청 사이트 v5

요청사항을 반영한 버전입니다.

## 변경된 기능

- 낙찰자 조회 후 낙찰 개체를 한 번에 체크 선택하는 방식 제거
- 낙찰 개체마다 `신청하기` 버튼 제공
- 버튼을 누르면 해당 개체 전용 `결제/수령 신청 페이지`로 이동
- 개체별로 결제 방식과 수령 방식을 따로 저장
- 결제 방식
  - 이체: 5% 할인
  - 카드: 추가/할인 없음
  - 피들: 3.3% 추가
- 수령 방식
  - 매장 방문수령: 방문 날짜 선택 + 5% 할인
  - 도도시: 맡기는 곳은 `렙타일갤러리-화성 봉담` 고정, 받는 곳은 손님 선택
- 도도시 선택창에서 예상 배송비 표시 제거
- 도도시 배송비는 최종 결제금액에 포함하지 않음
- `결제/수령 정보 등록 완료`를 누르면 저장 후 휴대폰 문자 앱으로 이동
- 문자 내용에는 성함, 연락처, 낙찰개체, 결제방식, 수령방법, 최종 결제금액, 입금계좌가 자동 입력됨
- 관리자 페이지에서 `신청 완료 문자 받을 번호` 설정 가능
- 손님 신청 페이지 낙찰자 조회는 등록일 기준 7일까지만 가능
- 관리자 페이지에서는 7일이 지나도 계속 조회/관리 가능
- CSV 다운로드에 개체별 신청 정보 포함

## 문자 발송 번호 설정

관리자 페이지 → 기본 설정 → `신청 완료 문자 받을 번호`에 매장 휴대폰 번호를 입력하세요.

예시:

```text
01044456632
```

기본값은 01044456632로 설정되어 있습니다. 관리자 페이지에서 다른 번호로 변경할 수 있습니다.

## 도도시 정거샵 목록

관리자 페이지에서 도도시 정거샵 목록을 수정할 수 있습니다. 형식은 아래처럼 한 줄에 하나씩 입력합니다.

```text
지역|받는곳명|주소|관리자용금액
경기A|깜크-안양|경기도 안양시 동안구 동편로39번길 17-6|30000
서울|티그리스게코[강동]|서울특별시 강동구 상암로 8-1|35000
```

현재 버전에서는 손님 선택창에 예상 배송비를 표시하지 않습니다. 마지막 금액 칸은 기존 데이터 호환과 관리자 참고용입니다.

## 파일 저장 주의

현재 버전은 `data/db.json` 파일 저장 방식입니다. Render 무료 웹서비스에서는 재시작/재배포 시 저장 데이터가 사라질 수 있으므로, 실제 장기 운영에는 DB 연결 버전을 권장합니다.

- 관리자 페이지에서 카카오톡 입력칸은 제거되었습니다.
