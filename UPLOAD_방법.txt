업로드 방법 - 중요

현재 Render 명령어는 아래처럼 되어 있어야 합니다.

Build Command:
npm install

Start Command:
npm start

이번 ZIP은 저장소 맨 위에 그대로 올리는 구조입니다.
압축을 풀면 바로 아래 파일들이 보여야 합니다.

server.js
package.json
public
data
README.md
UPLOAD_방법.txt

GitHub 저장소 맨 위 화면에서 Add file → Upload files로 위 파일/폴더를 모두 덮어올리세요.
ZIP 파일 자체를 올리면 안 됩니다.

배포 성공 로그 확인:
reptile-gallery-shipping@1.0.31 start
정적 파일 위치: /opt/render/project/src/public
렙타일갤러리 결제/수령 신청 사이트가 실행되었습니다. v31-root-direct-csv-fixed

관리자 페이지 확인:
https://reptile-gallery-shipping.onrender.com/admin.html

관리자 페이지 상단에 'v31 CSV 업로드 기능 적용됨'이 보이면 정상입니다.

데이터 영구 저장:
Render Environment에 DATA_DIR=/var/data
Render Disk Mount Path에 /var/data
