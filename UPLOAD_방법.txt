업로드 방법 - v32 내장파일 안전버전

이번 버전은 public 폴더가 GitHub 업로드 중 누락돼도 server.js 안에 내장된 파일로 사이트가 열리게 만든 버전입니다.

현재 Render 설정:
Build Command: npm install
Start Command: npm start

GitHub 저장소 맨 위에 아래 파일들을 덮어올리세요.
- server.js
- package.json
- public 폴더
- data 폴더
- README.md
- UPLOAD_방법.txt

가장 중요한 파일은 server.js와 package.json입니다.
public 폴더가 빠져도 server.js 내장 파일로 접속되게 해두었습니다.

성공 로그:
reptile-gallery-shipping@1.0.32 start
public 폴더가 빠져도 내장 파일로 화면을 표시합니다.
렙타일갤러리 결제/수령 신청 사이트가 실행되었습니다. v32-embedded-public-csv

확인 주소:
https://reptile-gallery-shipping.onrender.com/health
https://reptile-gallery-shipping.onrender.com/admin.html

관리자 페이지 상단에 'v32 CSV 업로드 기능 적용됨'이 보이면 정상입니다.
